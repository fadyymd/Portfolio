# 🚀 Fadi Medkour — Portfolio

> بورتفوليو شخصي احترافي بتصميم أزرق وأسود عصري

## 📁 هيكل المشروع

```
portfolio/
├── index.html          ← الصفحة الرئيسية
├── css/
│   └── style.css       ← كل الأنماط والتصميم
├── js/
│   └── main.js         ← التفاعلات والحركات
├── assets/
│   └── profile.jpeg    ← صورة البروفايل
└── README.md           ← هذا الملف
```

## 🛠️ التقنيات المستخدمة

| التقنية        | الاستخدام                              |
|----------------|----------------------------------------|
| **HTML5**      | بنية الصفحة والمحتوى                  |
| **CSS3**       | التصميم والأنيميشن والـ Grid           |
| **JavaScript** | التفاعل، الـ Observer، الـ Typing      |
| **Google Fonts** | خطوط Syne + Space Mono + Outfit    |

> ⚠️ هذا البورتفوليو مبني بـ Vanilla HTML/CSS/JS بدون أي Framework خارجي
